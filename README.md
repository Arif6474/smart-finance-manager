# TakaHisab

TakaHisab is a comprehensive personal finance tracking and management application. It helps users achieve financial clarity through smart tracking, automated budgeting, and AI-driven insights.

## 🚀 Key Features

-   **Multi-Account Tracking**: Manage Bank, Cash, and Mobile Banking accounts in one place.
-   **Transaction Management**: Record income, expenses, and internal transfers with real-time balance updates.
-   **Smart Budgeting**: Set monthly category-based spending limits and track progress.
-   **AI Financial Assistant**: Get personalized advice and insights powered by OpenAI/Gemini.
-   **Debt Management**: Track Payables (money you owe) and Receivables (money owed to you) with loan/EMI support.
-   **Subscription Tracking**: Stay on top of recurring payments.
-   **Financial Health Score**: Gamified scoring system to track your financial progress.
-   **Visual Reports**: Understand your spending patterns through interactive charts and analytics.

## 🛠️ Technology Stack

-   **Frontend**: Next.js 14, React, Tailwind CSS, Framer Motion
-   **Backend**: Next.js API Routes, Mongoose
-   **Database**: MongoDB
-   **AI**: OpenAI API / Google Generative AI
-   **Auth**: JWT, Google OAuth

## 🚦 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Arif6474/smart-finance-manager.git
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env.local` file based on `.env.local.example`.

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

5.  **Open the app**:
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
